**Button Component**
```jsx
import React from 'react';
import { View, Text, TouchableOpacity } from 'react-native';

const MyButton = () => {
  const pressHandler = () => {
    console.log('Button pressed!');
  };

  return (
    <TouchableOpacity
      style={{
        backgroundColor: '#007bff',
        padding: 10,
        borderRadius: 5,
        alignItems: 'center',
        justifyContent: 'center',
      }}
      onPress={pressHandler}
    >
      <Text style={{ color: '#fff', fontSize: 16 }}>
        Press me!
      </Text>
    </TouchableOpacity>
  );
};

export default MyButton;
```

**Explanation**

* We import the necessary components from `react-native`.
* We define a `MyButton` component that returns a `TouchableOpacity`.
* We define a `pressHandler` function that logs a message to the console when the button is pressed.
* We style the button with a background color, padding, border radius, and text color.
* We render a `Text` component with the button's text inside the `TouchableOpacity`.

**Example Use Case**

```jsx
import React from 'react';
import MyButton from './MyButton';

const App = () => {
  return (
    <View>
      <MyButton />
    </View>
  );
};

export default App;
```

This will render a small button with the text "Press me!" when the app is run.
