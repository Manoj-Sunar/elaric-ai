Here's an example of a small button in React Native:

```jsx
import React from 'react';
import { View, Text, TouchableOpacity, StyleSheet } from 'react-native';

const SmallButton = () => {
  return (
    <TouchableOpacity style={styles.button} onPress={() => console.log('Button pressed')}>
      <Text style={styles.buttonText}>Click me!</Text>
    </TouchableOpacity>
  );
};

const styles = StyleSheet.create({
  button: {
    backgroundColor: '#007bff',
    padding: 10,
    borderRadius: 5,
    alignItems: 'center',
    justifyContent: 'center',
  },
  buttonText: {
    color: '#ffffff',
    fontSize: 16,
  },
});

export default SmallButton;
```

In this code:

*   We import the necessary components from `react-native`.
*   We define a `SmallButton` component as a function component.
*   Inside the component, we return a `TouchableOpacity` component, which is a touchable button.
*   We style the button with `styles.button` and the text with `styles.buttonText`.
*   We add an `onPress` event handler to log a message to the console when the button is pressed.
*   We create a `styles` object using `StyleSheet.create()` to define the styles for the button and text.

You can use this component in your app by importing it and rendering it in your JSX:

```jsx
import React from 'react';
import SmallButton from './SmallButton';

const App = () => {
  return (
    <View style={{ flex: 1, justifyContent: 'center', alignItems: 'center' }}>
      <SmallButton />
    </View>
  );
};
```
