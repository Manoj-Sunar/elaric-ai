Here's an example of a beautiful React Native button:

```jsx
// Button.js
import React from 'react';
import { View, Text, StyleSheet, TouchableOpacity } from 'react-native';

const Button = ({ title, onPress }) => {
  return (
    <TouchableOpacity
      style={styles.button}
      onPress={onPress}
    >
      <Text style={styles.text}>{title}</Text>
    </TouchableOpacity>
  );
};

const styles = StyleSheet.create({
  button: {
    backgroundColor: '#4CAF50',
    padding: 15,
    borderRadius: 5,
    shadowColor: '#000',
    shadowOffset: { width: 0, height: 2 },
    shadowOpacity: 0.5,
    shadowRadius: 2,
    elevation: 2,
    width: '100%',
    alignItems: 'center',
    justifyContent: 'center',
  },
  text: {
    color: '#fff',
    fontSize: 18,
    fontWeight: 'bold',
  },
});

export default Button;
```

In this example, we're creating a reusable `Button` component that can be used throughout our app. The button is created using a `TouchableOpacity` component, which is a touchable component that handles the press event. The button's styling is defined using the `styles` object, which is a common pattern in React Native.

You can use this button in your app like this:

```jsx
// App.js
import React from 'react';
import Button from './Button';

const App = () => {
  const handlePress = () => {
    console.log('Button pressed!');
  };

  return (
    <View style={{ flex: 1, justifyContent: 'center', alignItems: 'center' }}>
      <Button title="Click me!" onPress={handlePress} />
    </View>
  );
};

export default App;
```

This will render a beautiful button with the title "Click me!" that you can press to log a message to the console.

To make the button even more beautiful, you can customize its styles and add animations, gradients, or other effects.
