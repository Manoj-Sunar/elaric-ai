Here is an example of a basic React Native button component:

```jsx
import React from 'react';
import { View, Text, TouchableOpacity, StyleSheet } from 'react-native';

const Button = ({ onPress, title }) => {
  return (
    <TouchableOpacity style={styles.button} onPress={onPress}>
      <Text style={styles.text}>{title}</Text>
    </TouchableOpacity>
  );
};

const styles = StyleSheet.create({
  button: {
    backgroundColor: '#007bff',
    padding: 15,
    borderRadius: 5,
    alignItems: 'center',
    justifyContent: 'center',
  },
  text: {
    color: '#ffffff',
    fontSize: 16,
  },
});

export default Button;
```

You can use this button component in your React Native app like this:

```jsx
import React from 'react';
import Button from './Button';

const App = () => {
  const handlePress = () => {
    console.log('Button pressed!');
  };

  return (
    <View style={{ flex: 1, justifyContent: 'center', alignItems: 'center' }}>
      <Button onPress={handlePress} title="Press me!" />
    </View>
  );
};

export default App;
```

This code creates a basic button with a blue background, white text, and a 5px border radius. When the button is pressed, it logs a message to the console.

You can customize the styles and behavior of the button by modifying the `styles` object and the `Button` component's props.
