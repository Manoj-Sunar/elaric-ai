**Button Component in React Native**
=====================================

Here's a simple example of a button component in React Native:

```jsx
// Button.js
import React from 'react';
import { View, Text, TouchableOpacity, StyleSheet } from 'react-native';

const Button = ({ title, onPress }) => {
  return (
    <TouchableOpacity style={styles.button} onPress={onPress}>
      <Text style={styles.text}>{title}</Text>
    </TouchableOpacity>
  );
};

const styles = StyleSheet.create({
  button: {
    backgroundColor: '#4CAF50',
    paddingVertical: 10,
    paddingHorizontal: 20,
    borderRadius: 5,
  },
  text: {
    color: '#fff',
    fontSize: 16,
    fontWeight: 'bold',
  },
});

export default Button;
```

**Usage**
-----

```jsx
// App.js
import React from 'react';
import Button from './Button';

const App = () => {
  const handlePress = () => {
    console.log('Button pressed!');
  };

  return (
    <View>
      <Button title="Click me!" onPress={handlePress} />
    </View>
  );
};

export default App;
```

In this example, we create a `Button` component that takes a `title` and an `onPress` function as props. The `TouchableOpacity` component is used to create a touchable button, and the `Text` component is used to display the button title. The styles are defined using the `StyleSheet.create` method.

You can customize the button's appearance and behavior by modifying the styles and the `onPress` function.
