Here's an example of a React Native button:

```jsx
import React from 'react';
import { View, Text, TouchableOpacity, StyleSheet } from 'react-native';

const Button = () => {
  return (
    <TouchableOpacity style={styles.button}>
      <Text style={styles.buttonText}>Click Me</Text>
    </TouchableOpacity>
  );
};

const styles = StyleSheet.create({
  button: {
    backgroundColor: '#4CAF50',
    padding: 10,
    borderRadius: 5,
    alignItems: 'center',
    justifyContent: 'center',
  },
  buttonText: {
    color: '#fff',
    fontSize: 16,
  },
});

export default Button;
```

This code defines a reusable `Button` component that can be used throughout your app. The button has a green background, white text, and is centered.

### Usage:

To use this button in your app, simply import it and render it like this:

```jsx
import React from 'react';
import Button from './Button';

const App = () => {
  return (
    <View style={{ flex: 1, justifyContent: 'center', alignItems: 'center' }}>
      <Button />
    </View>
  );
};

export default App;
```

You can customize the appearance of the button by modifying the styles in the `styles` object.

### Props:

You can pass props to the button to customize its behavior. For example, you can pass a `disabled` prop to disable the button:

```jsx
import React from 'react';
import Button from './Button';

const App = () => {
  return (
    <View style={{ flex: 1, justifyContent: 'center', alignItems: 'center' }}>
      <Button disabled={true} />
    </View>
  );
};
```

You can also pass a `onPress` prop to handle the press event:

```jsx
import React from 'react';
import Button from './Button';

const App = () => {
  const handlePress = () => {
    console.log('Button pressed!');
  };

  return (
    <View style={{ flex: 1, justifyContent: 'center', alignItems: 'center' }}>
      <Button onPress={handlePress} />
    </View>
  );
};
```
